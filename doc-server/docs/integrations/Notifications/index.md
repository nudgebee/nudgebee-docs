# Notifications

NudgeBee can send notifications through a variety of channels to keep you informed about important events and updates. This section details how to set up and configure the various notification integrations available.

## Available Notification Channels

NudgeBee currently supports the following notification channels:

*   **[Slack](./slack.md)**: Receive notifications directly in your Slack workspace.
    *   [Slack Setup in NudgeBee Account](./slack/#how-to-configure-slack-in-your-nudgebee-account)
    * [Slack setup in on-premise](./slack/#how-to-configure-slack-in-your-on-prem-nudgebee)
*   **[Microsoft Teams](./msteams.md)**: Integrate with Microsoft Teams to receive notifications in your team channels.
*   **[Google Chat](./google_chat.md)**: Get notifications in your Google Chat rooms.
