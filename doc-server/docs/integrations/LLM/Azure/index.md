# Azure Integration Options

This guide covers the integration options available for connecting your NudgeBee applications with Microsoft Azure AI services.

## Available Azure Integration Options

NudgeBee supports the following Azure services for LLM integration:

1. [Azure OpenAI Service](./azure-openai) - Access state-of-the-art language models with enterprise-level security and compliance
2. [Azure AI Custom Models](./nb-azure-ai) - Deploy and manage your custom models on Azure infrastructure


## Getting Started

To integrate with Azure services, you'll need:

1. An active Azure subscription
2. Access to Azure OpenAI Service (requires application)
3. Azure service principal or managed identity credentials
4. Resource group and region selection

Select the specific Azure service from the options above to begin the integration process.
