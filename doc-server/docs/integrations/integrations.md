# Integrations

NudgeBee integrates with a variety of tools and platforms to enhance your workflow and streamline your processes. This section provides an overview of our integrations and will be updated as more integrations become available.

## Available Integrations

### Observability
* [Observability Platform](./Observability/Observability.md) - Overview of all observability integrations.
* [Prometheus](./Observability/prometheus.md) - Metrics collection from Prometheus.
* [Loki](./Observability/loki.md) - Log querying from Grafana Loki.
* [Elasticsearch](./Observability/elasticsearch.md) - Logs, metrics, and traces from Elasticsearch / OpenSearch.
* [Jaeger](./Observability/jaeger.md) - Distributed tracing from Jaeger.
* [Datadog](./Observability/datadog.md) - Metrics, logs, and traces from Datadog.
* [New Relic](./Observability/newrelic.md) - Full-stack observability from New Relic.
* [Azure Application Insights](./Observability/azure_app_insights.md) - Monitoring from Azure Application Insights.

### Notifications
* [Slack](./Notifications/slack.md) - Real-time notifications to Slack channels.
* [MS Teams](./Notifications/msteams.md) - Real-time notifications to Microsoft Teams channels.

### Tickets
* [Jira](./Tickets/jira.md) - Issue tracking with Atlassian Jira.
* [GitHub Issues](./Tickets/github_issues.md) - Issue tracking with GitHub Issues.
* [ServiceNow](./Tickets/servicenow.md) - Incident management with ServiceNow.
* [PagerDuty](./Tickets/pagerduty.md) - Incident management with PagerDuty.

### CI/CD
* [ArgoCD](./CICD/argocd-integration.md) - GitOps continuous delivery with ArgoCD.

### Collaboration
* [Confluence](./Collaboration/confluence.md) - Knowledge base sync from Atlassian Confluence.

### Source Control
* [GitHub](./GitHub/github-integration.md) - Connect GitHub for code analysis and automated pull requests.
* [GitLab](./GitLab/gitlab-integration.md) - Connect GitLab for issue tracking and automated merge requests.

### Cloud
* [AWS](../features/Cloud/AWS.md) - Amazon Web Services integration.
* [Azure](../features/Cloud/Azure.md) - Microsoft Azure integration.
* [GCP](../features/Cloud/GCP.md) - Google Cloud Platform integration.
* [Cloud Foundry](../features/Cloud/CloudFoundry.md) - Cloud Foundry / Tanzu Application Service integration.

### Authentication
* [Authentication](./Authentication/) - Integrate NudgeBee with an existing authentication system. Only applicable for on-prem.

### AI / LLM
* [NudgeBee AI](./LLM/) - Integrate LLM models into your NudgeBee instance. Only applicable for on-prem.
