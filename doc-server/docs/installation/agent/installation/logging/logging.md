# Logging

Nudgebee can integrate with existing logging systems like Loki and Elasticsearch (ELK) to provide a unified view of your application logs directly within the Nudgebee UI. This enables you to search, filter, and analyze logs using the familiar query formats of Loki and Elasticsearch.

This section provides instructions on how to integrate Nudgebee with your existing logging infrastructure.

## Supported Logging Providers

Nudgebee supports integration with loki/open-search compatible logging providers.

*   **Elasticsearch (ELK Stack):** [Integrate with ELK Stack](./elk)
*   **Grafana Loki:** [Integrate with Grafana Loki](./loki)
*   **Logz.io:** [Integrate with Logz.io](./logz.io)
*   **Last9:** [Integrate with Last9](./last9)
