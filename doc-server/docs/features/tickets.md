# Tickets

NudgeBee integrates with popular ticketing systems so you can create tickets for errors, events, optimization recommendations, and log entries — either manually through the UI or automatically using [Autopilot](./autopilot/autopilot.md).

:::info
**Prerequisite**: To use ticketing features, first connect a [ticketing system integration](/docs/integrations/Tickets/) (Jira, ServiceNow, PagerDuty, GitHub Issues, or GitLab Issues).
:::

### Available Ticketing Systems

1. [Jira](/docs/integrations/Tickets/jira)
2. [GitHub Issues](/docs/integrations/Tickets/github_issues)
3. [GitLab Issues](/docs/integrations/Tickets/gitlab)
4. [ServiceNow](/docs/integrations/Tickets/servicenow)
5. [PagerDuty](/docs/integrations/Tickets/pagerduty)
6. [Zenduty](/docs/integrations/Tickets/zenduty)
