# Tickets


Integrate with popular ticketing systems so that tickets can be created for errors, events, recommendations and logs through UI or automate the creation and assignment of tickets based on pre-configured settings using AutoPilot.

Available ticketing systems for integration:

1. [Jira](/docs/integrations/Tickets/jira.md)
2. [Github Issues](/docs/integrations/Tickets/github_issues.md)
