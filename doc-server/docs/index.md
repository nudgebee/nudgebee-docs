# NudgeBee Documentation

NudgeBee is an **AI-agentic workflow platform** designed for SRE, CloudOps, and support teams to automate troubleshooting, cost optimization, and infrastructure management. 

By leveraging purpose-built AI assistants and a customizable workflow builder, NudgeBee helps organizations reduce operational complexity and optimize cloud infrastructure.
